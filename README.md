# devops-netology

Этот репозиторий содержит материалы, необходимые для обучения на курсе "Devops-инженер с нуля: расширенный курс" от Нетологии.

## Terraform .gitignore

В каталоге terraform добавлен файл .gitignore, который исключает из репозитория служебные файлы Terraform. 

В частности будут игнорироваться:

- каталог .terraform с загруженными провайдерами
- файлы состояния terraform.tfstate
- файлы переменных *.tfvars
- файлы переопределения конфигурации override.tf
- файл блокировки .terraform.lock.hcl

Это позволяет не хранить в репозитории временные и служебные файлы.

Ветка fix создана от коммита Prepare to delete and move.
